# data-field-guides

Field guides for data practitioners who want to do the work properly. 
Structured approaches to data quality, analysis, and output, distilled 
from years working with data across Fortune 500 enterprises, international 
organisations, and UN agencies.

## What you'll find here

| Guide | Description | Status |
|-------|-------------|--------|
| tidy-data-companion | A structured diagnostic and cleaning guide for messy datasets. Outputs a cleaned file, data dictionary, diagnostic narrative, and change log. | Stable |

## How to use

These guides are built as structured skills in SKILL.md format — 
compatible with Claude, Cursor, Gemini CLI, and other AI assistants 
that support the open skills standard. Each guide folder contains 
the skill file and its own README with setup and usage notes.

## More coming

This repo grows as the work does. Guides in the pipeline cover dashboard 
frameworks, data communication, and more.
